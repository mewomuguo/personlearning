## 第三章:技術憲章與踩坑史(新 AI 必讀)

1. **babel-standalone 以 `new Function` 執行**:跨 `<script type="text/babel">` 區塊的 const 不共享 → app.js 必須是單一 babel 檔;data.js 用普通 script(全域詞法環境可被讀取)
2. **本機雙擊 index.html 不能跑**(babel XHR 被 file:// 擋)→ `python3 -m http.server`;Pages 上正常
3. **快取紀律**:每次更新內容,`CONTENT_VERSION` 與 index.html 的 `?v=N` 必須同步遞增,否則 GitHub Pages/瀏覽器供應舊檔
4. **上傳紀律**:大檔必須用 GitHub「Upload files」拖曳上傳,**禁止複製貼上內容**(手機剪貼簿會截斷,曾造成 data.js 斷在 355 行的全站白畫面)
5. **React hooks 引入**:新元件用到新 hook(如 useRef)必須加進頂部 destructure——曾漏 useRef 造成進遊戲即崩潰
6. **相容性**:`structuredClone` 已有 polyfill(舊 Safari);動畫全部尊重 `prefers-reduced-motion`
7. **字串守則**:單引號 JS 字串內禁用撇號與雙引號;模板字串(反引號)內可用
8. **JSX 無法 node --check**:data.js 可以、app.js 不行;app.js 改動後至少做括號平衡 + 抽取純 JS 邏輯用 node 單元測試(路徑插值、生成器煙霧測試皆有先例)

**驗證紀律(每次修改後必跑)**:
1. `node --check js/data.js`(語法)
2. 括號平衡:python 檢查 app.js 的 `{}` `()` `[]`
3. 內容驗證腳本(附錄 A):字數/重複/跨級重疊/欄位完整/詳解結構
4. 出題引擎煙霧測試(附錄 B):每生成器 ×300 次,查 undefined/NaN/選項重複/雙語缺失

## 第四章:資料格式規格(擴充只碰這些)

### 漢字 `HANZI[level] = [字族]`
```js
{f:'字族名', fe:'English', rule:'破解規則(中文)', re:'Rule in English',
 chars:[{c:'字', zy:'ㄓㄨˋ', en:'meaning', w:'詞語', we:'word'}]}
```
守則:注音依教育部《國語辭典簡編本》(台陸異讀以台灣為準,實例:驟ㄗㄡˋ、夕ㄒㄧˋ、熟ㄕㄡˊ、寂ㄐㄧˊ);輕聲 `˙ㄕㄤ`;英文禁撇號雙引號;rule 必須是可遷移邏輯;跨級去重。

### 自然 `SCI[level] = [單元]`
```js
{title, concept, en, qs:[{q, en, options:[×4], ans, why:[四段陣列]}]}
```
why 四段:`'🔑 推理規則…'` → `'① 證據…' '② 推論…'` → `'✔ 驗證…'` 或 `'💡 遷移…'` → `'EN: …'`

### 數學 `MATH_GEN[level] = [生成器函式]`(app.js)
每函式回傳 `mc(q, en, answer, distractors, whyArray)`;mc 已內建選項去重與數值補位。奧數題以 🏆 開頭。`MATH_TOPICS` 同步維護題型地圖。輔助函式:`ri(a,b)` 隨機整數、`pick`、`shuffle`、`nearNums(ans,spread)`。

### 存檔 schema(勿改 key 名)
```js
{xp, totalXp, seals:[], streak, lastDay,
 pets:[id], activePet, petData:{id:{lv,bond,sprints}},
 best:{match,storm,defense},
 farm:{v:2, day, coins, energy, harvested, tiles:[{crop,stage,watered}|null ×24]},
 subjects:{hanzi:{level},math:{level},science:{level}}, engHints}
```
新增欄位一律走 DEFAULT 合併(向後相容);農場已示範 v1→v2 遷移模式。

## 第五章:學科擴充路線圖(9 → 18 歲)

### 5.1 國語文:漢字 → 詞彙 → 文言 → 寫作

| 年齡 | 內容 | 完成證據 |
|---|---|---|
| 9–10(現在) | 千字文 1031 字 ✅ → **部首家族包**(氵/心/扌/口/木…每包 40–60 字,沿用字族格式),直通教育部 2400 常用字 | 星圖點亮數;每包結業硃砂印 |

> 大原則:**動機系統要跟著長大,但告別童年元素的時機由使用者本人決定。** 強行「轉大人」摧毀信任;永不更新則系統變幼稚而被棄用。

|---|---|---|---|

## 附錄 A:內容驗證腳本(每次改 data.js 必跑)

```python
import re
data=open('js/data.js').read()
assert data.rstrip().endswith('const DATA_OK=true;'),'哨兵必須是最後一行'
def lv(txt,n):
    s=txt.index(f'\n  {n}:[')
    e=[txt.index(f'\n  {m}:[') for m in range(n+1,7) if f'\n  {m}:[' in txt]
    return re.findall(r"\{c:'(.)'",txt[s:min(e+[txt.index('const SEAL_CHARS')])])
seen={}
for n in range(1,7):
    cs=lv(data,n);d=[c for c in set(cs) if cs.count(c)>1]
    print(f'L{n}: {len(cs)} 字 重複:{d or "無"}')
    for c in cs:
        assert c not in seen,f'跨級重複 {c} L{seen[c]}/L{n}';seen[c]=n
print('完整字卡:',len(re.findall(r"\{c:'.',zy:'[^']+',en:'[^']+',w:'[^']+',we:'[^']+'\}",data)))
```
